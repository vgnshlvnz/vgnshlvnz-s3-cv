# Profile Image

Add your professional profile photo here as `profile.jpg`.

## Image Requirements

- **Filename**: Must be named exactly `profile.jpg`
- **Format**: JPEG format preferred
- **Size**: Recommended 400x400 pixels (square)
- **Quality**: High quality, professional photo
- **File Size**: Keep under 500KB for fast loading

## Image Guidelines

### Professional Appearance
- Business or business casual attire
- Clean, well-groomed appearance
- Professional lighting
- Neutral or professional background

### Technical Specifications
- **Aspect Ratio**: 1:1 (square) works best
- **Resolution**: At least 400x400px, max 800x800px
- **Format**: JPEG (.jpg) or PNG (.png)
- **Color**: Full color preferred
- **Compression**: Balance quality vs file size

### Alternative Options
If you don't have a professional photo:
- Use a professional headshot service
- Take a photo with good lighting against a plain wall
- Use a high-quality smartphone camera
- Consider using a placeholder avatar temporarily

## CSS Styling
The image is styled with:
- Circular crop (`border-radius: 50%`)
- White border for contrast
- Drop shadow for depth
- Responsive sizing for different screen sizes

## Placeholder
If no image is provided, you may want to add a placeholder or default avatar image.
