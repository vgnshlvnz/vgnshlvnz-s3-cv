# Add your CV files here

Please add your CV files with these exact names:

1. `Vigneshwaran_Ravichandran_CV.pdf` - PDF version of your CV
2. `Vigneshwaran_Ravichandran_CV.docx` - Word document version of your CV

## File Requirements

- **PDF**: Should be optimized for web viewing
- **Word**: Should be compatible with recent versions of Microsoft Word
- **File Size**: Keep under 5MB for better download experience
- **Content**: Ensure both files contain the same information

## Security Note

These files will be publicly accessible when deployed to S3. Make sure:
- No sensitive personal information (like full address, SSN, etc.)
- Only professional contact information
- Current and accurate information

## File Naming Convention

The JavaScript expects these exact filenames. If you change them, update the `downloadCV()` function in `../js/script.js` accordingly.
